# Local Skills

This directory contains locally installed skills that are specific to this project.

## Available Skills

### nestjs-expert
Specialized NestJS backend development expertise for building scalable, enterprise-grade TypeScript applications.

### nextjs-developer
Specialized Next.js frontend development expertise for building modern React applications with App Router, Tailwind, and related technologies.

### postgres-pro
Expert PostgreSQL database administration and optimization skills for designing efficient schemas, writing optimized queries, and managing database performance.

### testing-wizard
Comprehensive testing expertise covering unit, integration, and end-to-end testing strategies to ensure code quality and reliability. (Based on test-master skill)

## Usage

These skills can be invoked using the Skill tool:
- `/nestjs-expert` - For NestJS backend development tasks
- `/nextjs-developer` - For Next.js frontend development tasks
- `/postgres-pro` - For PostgreSQL database tasks
- `/testing-wizard` - For testing-related tasks

Skills are automatically available when working within this project directory.
